​EcoLearn: Gamified Environmental Education Platform

​EcoLearn is a user-friendly web application designed for schools and colleges to promote environmental awareness through gamified learning. It encourages students to build sustainable habits by participating in eco-friendly activities, learning about environmental topics, and earning rewards.

​🚀 Key Features
​Student Module: Secure registration, login, and a personalized dashboard to track eco-points and activity progress.
​Learning Module: Interactive lesson cards on topics like Climate Change, Waste Management, Water Conservation, Energy Saving, and Biodiversity.
​Quiz Module: Multiple-choice quizzes for each topic that reward students with eco-points upon completion.
​Eco Challenge Module: Real-world daily activities (e.g., planting trees, saving water) that users can mark as completed to earn points automatically.
​Leaderboard & Rewards: A competitive leaderboard ranking students, with digital badges like "Eco Warrior" and "Green Champion".
​Admin Module: Dedicated access for administrators to manage lessons, quizzes, challenges, and view student progress.

​🛠️ Technology Stack
​This project was built with a focus on responsiveness and performance:
​Frontend: HTML, CSS, JavaScript
​Backend: Firebase / Supabase
​Responsive Design: Optimized for both mobile and desktop devices.

​🎯 Project Objective
​The goal of EcoLearn is to bridge the gap between environmental awareness and active participation. By gamifying the learning process, we aim to turn sustainable habits into an engaging daily routine for students.
